Text Expander Help

Learn how to use the Chrome Text Expander extension

Getting Started

How It Works

Text Expander allows you to type short abbreviations that automatically expand into longer text. For example, you can type sig: and it will automatically expand to your full signature.

Creating Your First Snippet

  1. Click the Text Expander icon in your Chrome toolbar
  2. Click "Add New" button
  3. Enter a shortcut (e.g., "sig")
  4. Enter the text you want it to expand to
  5. Optionally assign a category
  6. Click "Save"

Using Snippets

To use a snippet, simply type its shortcut followed by the trigger character (default is :) in any text field. The shortcut will be immediately replaced with your expanded text.

Type: sig:
Gets replaced with: Best regards,
Your Name

Advanced Features

Customizing the Trigger Character

You can change the trigger character (default is :) in the extension settings:

  1. Click the gear icon in the Text Expander popup (or right-click the extension icon and select "Options")
  2. In the "Trigger character" field, enter your preferred character
  3. Click "Save Settings"

For example, if you change the trigger character to ;, you would type sig; to expand your signature.

Using Variables

You can include special variables in your snippets that will be replaced with dynamic content:

Variable Description Example
%DATE% Current date 3/20/2025
%TIME% Current time 2:30 PM
%DATETIME% Current date and time 3/20/2025, 2:30 PM
%CLIPBOARD% Current clipboard content [Whatever is in your clipboard]
%CURSOR% Where cursor should be placed after expansion N/A

Organizing Snippets

You can organize your snippets using categories. To filter snippets by category, use the dropdown menu at the top of the popup.

Backup and Restore

You can backup your snippets or transfer them to another computer using the Backup & Restore functionality:

  1. Click the "Backup & Restore" button at the bottom of the extension popup (or in Settings)
  2. A dialog will appear with two tabs:
    • Export Snippets: Copy the JSON data and save it to a file on your computer
    • Import Snippets: Paste previously exported JSON data to restore your snippets

Note: Importing will replace all your current snippets, so make sure to back up your existing snippets before importing new ones.

Troubleshooting

Snippets Not Expanding

  • Make sure the extension is enabled in your settings
  • Try refreshing the page
  • Check that you're typing the correct shortcut followed by your trigger character
  • Some websites may block content scripts - try a different site

Issues with Backup & Restore

  • Make sure your JSON data is properly formatted
  • When copying exported data, ensure you select all the text completely
  • If import fails, check for syntax errors in your JSON data

Other Issues

If you encounter any other issues, try these steps:

  1. Disable and re-enable the extension
  2. Check for conflicts with other extensions
  3. Clear your browser cache
  4. Reinstall the extension

Privacy

Text Expander respects your privacy:

  • All snippets are stored locally in your browser
  • No data is sent to any server
  • Your snippets are only accessible by you
  • The extension only requires minimal permissions to function
